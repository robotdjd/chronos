Mount driveNext, mount the drive to make it available to the file system on your Raspberry Pi:
$ sudo mount /dev/sda1
And ensure that the drive is mounted after each boot:
$ sudo nano /etc/fstab
Add the following line at the end of the file:
/dev/sda1 /mnt/sda1/ ext4 defaults,noatime 0 1
Press Ctrl+X, then Y, and finally Enter to save the edited file with nano.
